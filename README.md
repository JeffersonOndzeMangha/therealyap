# YAP – Yet Another Podcast

The official website for the [YAP YouTube channel](https://www.youtube.com/@therealYAP.official).

## Tech Stack

- **[Vite](https://vite.dev/)** – lightweight build tool and dev server (vanilla JS)
- **[Vitest](https://vitest.dev/)** – unit testing framework
- **[Vercel](https://vercel.com/)** – hosting and deployment

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build locally
npm run preview
```

## Testing

```bash
# Run tests once
npm test

# Run tests in watch mode
npm run test:watch

# Run tests with coverage
npm run test:coverage
```

## Deployment

The site is configured for automatic deployment on [Vercel](https://vercel.com/) via `vercel.json`. Any push to the main branch will trigger a new deployment.
